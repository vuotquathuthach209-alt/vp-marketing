📘 Facebook Fanpages

Thêm Fanpage mới

❓ Cách lấy Page Access Token
  1. Vào Graph API Explorer
  2. Chọn Application, chọn User Token
  3. Tick quyền: pages_show_list, pages_manage_posts, pages_read_engagement
  4. Bấm "Generate Access Token" → đăng nhập FB
  5. Gọi /me/accounts để lấy token của từng Page
  6. Copy access_token và id của page dán vào đây

Xem HUONG-DAN.md trên VPS để biết chi tiết đổi token sang long-lived (2 tháng).
