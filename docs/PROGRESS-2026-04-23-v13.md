## 🔬 Bot bây giờ BIẾT nó đang đúng/sai — đây là bước quan trọng nhất để tự tiến hoá

Với schema này, các insight sau tự động emerge sau ~1 tuần traffic thật:

1. **Top reply_sources cao conversion** → expand, nhân rộng
2. **Top reply_sources cao misunderstood** → rewrite prompt hoặc add RAG content
3. **Stage drop-off pattern** → biết stage nào đuổi khách (ví dụ BUDGET_ASK drop 40% → câu hỏi budget đang khó chịu)
4. **Ghosted rate** → baseline để đo improvement
5. **Rage quit triggers** → tin nhắn nào chạm nút nóng của khách

Admin có thể nhìn dashboard → gắn label `bad` + viết `corrected_reply` → vòng lặp supervised fine-tuning sẵn sàng cho giai đoạn 2.
